---
title: setupVariableControls
description: Fonction qui permet l'ajout de variables dans le modèle d'email
---

## Démarrer

Ajoutez la fonction suivant en bas de votre modèle d'e-mail

```tsx
setupVariableControls(Untitled, {
  title: {
    label: "Titre",
    defaultValue: "Untitled",
    placeholder: "Title",
  },
})
```

## Props

<ResponseField name="label" type="string" required>
  Label de la variable affiché sur l'interface d'édition
</ResponseField>

<ResponseField name="type" type="string" required>
  Type of variable used only to generate random values for preview mode.  
  You can use one of those value:
  - any
  - companyName
  - fullName
  - mailAddress
  - phoneNumber
  - postalAddress
  - url
  - date
  - time
</ResponseField>

<ResponseField name="defaultValue" type="string">
  Specify the target attribute for the button link
</ResponseField>

<ResponseField name="placeholder" type="string">
  Specify the target attribute for the button link
</ResponseField>
