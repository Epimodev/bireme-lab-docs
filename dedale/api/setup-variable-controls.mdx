## Make variables editable

Dedale gives the possibility to replace variables by real data to preview your template.  
To be able to make react Props editable from variables panel, you have to use `setupVariableControls` like this:

```tsx
import { /* Components needed for my template */ } from "@react-email/components";
import React from "react";
import { setupVariableControls } from "@bireme-lab/dedale";

type Props = {
  name: string
}

const MyTemplate: React.FC<Props> = ({ name }) => {
  // My template code
}

setupVariableControls(MyTemplate, {
  name: {
    label: "Name",
    type: "fullName",
    placeholder: "Antoine",
    defaultValue: "Antoine",
    description: "User name",
  },
})

export default MyTemplate
```

Once you saved the template, you can open variables panel to generate ramdom values or edit manually variable.

## Parameters

<ResponseField name="label" type="string" required>
  Label displayed above the input in variable panel
</ResponseField>

<ResponseField name="type" type="string" required>
  Type of variable used only to generate random values for preview mode.  
  You can use one of those value:
  - any
  - companyName
  - fullName
  - mailAddress
  - phoneNumber
  - postalAddress
  - url
  - date
  - time
</ResponseField>

<ResponseField name="defaultValue" type="string">
  Value displayed in preview if the input is empty in variable panel.
</ResponseField>

<ResponseField name="placeholder" type="string">
  Input placeholder in variable panel
</ResponseField>

<ResponseField name="description" type="string">
  Small description displayed below input
</ResponseField>
