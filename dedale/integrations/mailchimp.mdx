[Mailchimp](https://mailchimp.com/) est une plateforme de marketing populaire qui permet aux entreprises de gérer et d'optimiser leurs campagnes d'e-mailing. Conçue pour les utilisateurs de tous niveaux, Mailchimp offre une gamme d'outils et de fonctionnalités autour des problématiques d'e-mailing.

Sur cette page nous allons voir ensemble comment exporter un modèle d'e-mail de Dédale vers la plateforme Mailchimp.

Vous pouvez également consulter la [documentation officielle de Mailchimp pour importer un modèle HTML personnalisé](https://mailchimp.com/fr/help/import-a-custom-html-template/)

<Warning>

Sur Mailchimp, l'édition de code personnalisé n'est disponible qu'à partir du plan Standard. Autrement vous ne pourrez pas importer de modèle d'email personnalisé.

</Warning>

## Exporter le modèle depuis Dédale

<Info>Suivez notre guide à propos des options d'export disponible sur notre [page dédiée au démarrage rapide](/dedale/demarrage-rapide)</Info>

## Créer un nouveau modèle dans Mailchimp

Rendez-vous sur votre espace Mailchimp et accéder aux options de création de modèle depuis l'onglet `Contenu` > `Modèles d'e-mails`. Puis cliquez sur le bouton `Créer un modèle`

<img
  className="block"
  src="/images/create-mailchimp-template.webp"
  alt="Créer un nouveau modèle dans Mailchimp"
/>

Une nouvelle page va s'ouvrir dans laquelle vous aurez plusieurs options disponibles, cliquez sur l'onglet `Code personnalisé`

<img
  className="block"
  src="/images/custom-code.webp"
  alt="Rendez-vous sur l'onglet code personnalisé"
/>

## Coller le code HTML dans Mailchimp

Sélectionnez l'option `Coller dans le code`.

<img
  className="block"
  src="/images/paste-code.webp"
  alt="Sélectionnez l'option Coller dans le code"
/>

<Info>

Pour le moment, l'édition du code `HTML` n'est disponible que part le `créateur classique` de Mailchimp.

</Info>

Nommez votre nouveau modèle et validez en cliquant sur `Enregistrer`.

<img
  className="block"
  src="/images/name.webp"
  alt="Nommez votre nouveau modèle et validez en cliquant sur Enregistrer"
/>

Vous allez être redirigé vers l'inteface d'édition classique de Mailchimp sur deux panneaux, à gauche vous aurez la prévisualisation et à droite le code du modèle.

Pour importer le modèle créé sur Dédale, reprenez l'étape [Exporter le modèle depuis Dédale](/dedale/integrations/mailchimp#exporter-le-modele-depuis-dedale) en sélectionnant l'option `Copier au format HTML`.
Une fois dans votre presse-papier, retournez sur l'éditeur de Mailchimp et remplacer le code par celui que vous venez de copier sur Dédale.

<img
  className="block"
  src="/images/replace.webp"
  alt="Remplacez le code à droite pas celui que vous avez exporter en choisissant l'option Copier au format HTML depuis Dédale"
/>

Vérifiez dans la partie prévisualisation que le modèle est conforme à vos attentes (structure, images etc.) et sauvegardez votre modèle en cliquant sur `Save` en dessous de l'éditeur de code.

<img
  className="block"
  src="/images/save-template.webp"
  alt="Sauvegardez votre modèle en cliquant sur Save en dessous de l'éditeur de code"
/>

## Importer un fichier ZIP

Une seconde option est disponible dans Mailchimp pour créer un modèle personnalisé à partir de code `HTML` : importer un fichier `ZIP`.

Cette seconde option permet d'inclure les images dans l'import, Mailchimp s'occupera de les stockers et de les afficher dans vos e-mails sans que vous n'ayez besoin de les héberger vous-même.

Sélectionnez la deuxième option d'export depuis Dédale : `Télécharger le modèle` pour récupérer le fichier `HTML` comportant le code du modèle.

Vous pouvez alors créer une archive `ZIP` qu'il vous faudra téléverser dans Mailchimp après avoir sélectionné l'option `Importer le fichier ZIP`.

<img
  className="block"
  src="/images/zip.webp"
  alt="Sélectionnez l'option Importer le fichier ZIP"
/>

## Utilisez votre modèle dans un e-mail

Retournez sur la page `Modèles` et cliquez sur la flèche à côté du modèle que vous venez de créer puis choisissez l'option `Créer un e-mail`.

<img
  className="block"
  src="/images/create-email.webp"
  alt="Sélectionnez l'option Importer le fichier ZIP"
/>

Pour plus d'informations concernant l'usage d'un modèle avec Mailchimp nous vous invitons à consulter la [documentation officielle](https://mailchimp.com/fr/help/save-use-email-template/).
