## Create an email template

To create a new template, click on the "+ Create" button.

<img
  className="block"
  src="/images/create.webp"
  alt="To create a new template, click on the + Create button."
/>

An empty `Untitled` project will be added to the list of templates. You can open the `Code` panel to start editing the code.

## Rename an email template

You can rename the model from its name in the bar at the top of the interface.

<img
  className="block"
  src="/images/rename.webp"
  alt="Renommer le modèle"
/>
