## Étape 0 : Comprendre l'interface Dédale

Dédale a été conçue pour rendre disponibles les fonctionnalités importantes de manière intuitive.
Nous avons pensé la navigation sous forme de panneaux, accessibles via la barre d'outil présente à droite de l'interface.

Dans l'ordre de haut de en bas vous trouvez les panneaux suivants :
- **Variables** : qui permet de modifier les valeurs des variables (dont nous parlerons )
- **Code**
- **Informations**

<Steps>
  <Step title="Créer mon premier modèle d'e-mail">
    Créer un nouveau modèle sur Dédale.
  </Step>
  <Step title="Ajouter des variables et prévisualiser mon modèle">
    Comprendre le fonctionnement des variables et du panneau correspondant
  </Step>
  <Step title="Envoyer un e-mail pour tester le résultat">
    Vérifier l'intégrité de de mon modèle dans un client comme Gmail ou Outlook
  </Step>
  <Step title="Exporter mon modèle vers mon service d'envoi d'e-mails">
    Apprendre comment intégrer mon modèle dans un outil comme [Mailchimp](/dedale/integrations/mailchimp) ou [Hubspot](/dedale/integrations/hubspot)
  </Step>
</Steps>

## Étape 1 : Créer mon premier modèle d'e-mail

Pour créer un nouveau modèle, cliquez sur le bouton "+ Créer".

<img
  className="block"
  src="/images/create.webp"
  alt="Pour créer un nouveau modèle, cliquez sur le bouton + Créer"
/>

Un projet vide `Untitled` va s'ajouter dans la liste des modèles. Vous pouvez ouvrir le panneau `Code` pour commencer à éditer le code.

### Renommer le modèle

Vous pouvez renommer le modèle à partir de son nom présent sur la barre en haut de l'interface.

<img
  className="block"
  src="/images/rename.webp"
  alt="Renommer le modèle"
/>

### Éditer le code

Tout ce qui concerne le développement de modèle se base sur [React](https://react.dev), [TypeScript](https://www.typescriptlang.org/) et sur la librairie [React Email](react.email) qui apporte une collection de composants non stylisés.

Nous vous invitons à parcourir [la documentation de React Email](https://react.email/docs) pour en apprendre davantage sur le développement d'un modèle d'e-mail avec React et TypeScript.


## Étape 2 : Ajouter des variables et prévisualiser mon modèle

Jetons un oeil au code généré par défaut lors de la création du modèle, et en particulier la section `addVariables` (pour accéder au code du modèle ouvrez le panneau `Code`).

```tsx
addVariables(Untitled, {
  title: {
    label: "Titre",
    defaultValue: "Untitled",
    placeholder: "Title",
  },
})
```

## Étape 3 : Envoyer un e-mail pour tester le résultat

Vous pouvez tester votre modèle dans votre client e-mail favoris via l'option "Envoyer un test" disponible en haut à droite de l'interface de Dédale.

<img
  className="block"
  src="/images/send-test.webp"
  alt="Option Envoyer un test"
/>

Ajoutez l'adresse e-mail du destinataire ainsi que l'objet et cliquez sur "Envoyer".

<img
  className="block"
  src="/images/test-dialog.webp"
  alt="Envoyer un e-mail pour tester le résultat"
/>

## Étape 4 : Exporter mon modèle vers mon service d'envoi d'e-mails

Pour exporter votre nouveau modèle il vous suffit de cliquer sur le bouton `exporter` disponible en haut à droite de l'interface. Vous aurez le choix entre :
- **Copier au format HTML :** pour copier le code `HTML` du modèle dans votre presse-papier,
- **Télécharger le modèle :** pour télécharger un fichier `HTML` contenant le code du modèle.
