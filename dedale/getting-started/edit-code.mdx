## Éditer le code

Tout ce qui concerne le développement de modèle se base sur [React](https://react.dev), [TypeScript](https://www.typescriptlang.org/) et sur la librairie [React Email](react.email) qui apporte une collection de composants non stylisés.

Nous vous invitons à parcourir [la documentation de React Email](https://react.email/docs) pour en apprendre davantage sur le développement d'un modèle d'e-mail avec React et TypeScript.
