## Overview of Dédale interface

Dédale has been designed to make important features available in an intuitive way.

Here is an illustrated overview of how the interface is working:

<img
  className="block"
  src="/images/dedale-interface.webp"
  alt="Understanding the Dédale interface"
/>

## <Icon icon="circle-1" size={24} iconType="solid" /> Sidebar

<img
  className="block"
  src="/images/dedale-interface-sidebar.webp"
  alt="Sidebar"
/>

**A:** Create a new template  
**B:** Trigger bulk edit mode and select templates to delete  
**C:** Select how you would like your templates list to be sorted  
**D:** List of templates  

## <Icon icon="circle-2" size={24} iconType="solid" /> Preview

<img
  className="block"
  src="/images/dedale-interface-preview.webp"
  alt="Preview"
/>

**A:** Actual preview canvas width and height with dropdown including fast responsive options  
**B:** Preview canvas with draggable sidebars to modify responsive values  
**C:** Show or hide options bar

## <Icon icon="circle-3" size={24} iconType="solid" /> Title bar

<img
  className="block"
  src="/images/dedale-interface-titlebar.webp"
  alt="Title bar"
/>

**A:** Export template. [Learn more about export](/dedale/getting-started/export)  
**B:** Send a test email. [Learn more about sending tests](/dedale/getting-started/send-a-test)  
**C:** Last modified at information

## <Icon icon="circle-4" size={24} iconType="solid" /> Tool bar

<img
  className="block"
  src="/images/dedale-interface-toolbar.webp"
  alt="Toolbar"
/>

**A:** Toggle Variables panel. [Learn more about variables](/dedale/getting-started/variables)  
**B:** Code panel. [Learn more about code edition](/dedale/getting-started/edit-code)  
**C:** Can I email? panel. [Learn more about Can I email? integration](/dedale/getting-started/can-i-email)  
**D:** Template informations panel
